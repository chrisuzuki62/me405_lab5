Detailed Description

Class for a proportional controller.

This class contains the approprate methods to set the gain, set the desired position, run the proportional controller, run a step response for a motor, and print out the data from the step response.

Constructs a controller object.

Establishes the initial values used by the methods in this class. The proportional gain is initially set to zero. There is an empty list used to collect encoder data. Variables to keep track of time are also established

Runs one instance of the proportional controller.

Calculates the error between the current position and the desired position. This error is multiplied by a proportional gain to get an input for the plant.
